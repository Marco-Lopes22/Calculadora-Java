# 🧮 Calculadora em Java

Projeto desenvolvido como parte dos meus estudos em Análise e Desenvolvimento de Sistemas para praticar lógica de programação e fundamentos de Java.

## 🚀 Funcionalidades
- ➕ Soma  
- ➖ Subtração  
- ✖ Multiplicação  
- ➗ Divisão  
- ⚠ Tratamento para divisão por zero  
- 🔁 Repetição de operações pelo usuário  

## 📚 Conceitos praticados
- Estruturas condicionais com `switch`
- Laços de repetição com `while`
- Entrada de dados com `Scanner`
- Variáveis e operadores
- Tratamento básico de erros
- Lógica de programação

## 🛠 Tecnologias utilizadas
- Java  
- VS Code  
- Git e GitHub  

## ▶ Como executar

Compile o projeto:

```bash
javac Calculadora.java
```

Execute:

```bash
java Calculadora
```
